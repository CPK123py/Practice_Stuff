Customization Overview

How Bootstrap Customization Works:
  1. Override Bootstrap variables in _variables.scss before importing Bootstrap
  2. Customize color variables: $primary, $secondary, $success, etc.
  3. Adjust spacing variables: $spacer, $font-sizes, $border-radius
  4. Recompile Sass using: npm run sass
  5. Custom compiled CSS includes your modifications

CSS Custom Properties (Variables)

This page uses CSS custom properties for color customization:

:root {
  --bs-primary: #667eea;
  --bs-secondary: #764ba2;
}

Compiled Sass Structure

Bootstrap Sass Directory Structure
scss/
├── _variables.scss (customize colors, sizes, etc.)
├── _mixins.scss (reusable Sass mixins)
├── _root.scss (CSS custom properties)
├── components/ (button, card, form styles)
├── bootstrap.scss (main entry point)
└── bootstrap.min.scss (minified version)

Key Customization Variables

Variable Description Default Value
$primary Primary brand color #0d6efd
$secondary Secondary color #6c757d
$spacer Base spacing unit (1rem) 1rem
$border-radius Default border radius 0.375rem
$font-family-base Default font family system-ui
